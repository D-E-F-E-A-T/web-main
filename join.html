Get Involved. Have fun!

The Orchidware Project is seeking inquisitive individuals for the following roles:

  • Java Backend Developer
  • AWS Cloud Developer
  • iOS Graphics Developer
  • Android Graphics Developer
  • Graphic Artists
  • C++ Developer
  • QA Testers
  • Story Designer
  • Web Developer
  • Mathematician
  • Chemist
  • Biologist
  • Physicist

While these are unpaid roles, development hardware is available for checkout.

For Developer roles, it is very helpful to have a computer science, engineering, or math background; however, all skill levels are welcome. Those who have never written a line of code, but are persistent and patient, are welcome.

To allow individuals to experience different disciplines, Developer roles are available on a seasonal basis: Winter, Spring, Summer, Fall. Developers are encouraged to explore other development roles seasonally.

If you'd like to join, please send us a note.
