Classroom Challenge

For Students and Teachers: Classroom Challenge allows students and teachers to
make fun, exciting, and engaging class presentations.


For Software Developers:
Classroom challenge is a distributed software system which uses REST-ful
web services for client-server communication.
The system is composed of a set of backend services and several client user interface implementations
Each User Interface implements the same set of core features allows one to compare and contrast implementations
amongst the various user-interface technologies.
The UI technologies will include the following: Android, iOS, Javascript, Windows Store App API, and MFC. It uses the following technologies:
Technology Purpose
Java 1.8
Spring Framework Creating REST web services
JPA Persisting data to the database using Plain-Old-Java-Objects(POJO)
javascript For creating web user-interfaces. Angular.js
Jackson Converting POJO objects to JSON
Spring Tool Suite
